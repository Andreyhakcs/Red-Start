# Red NeuroShop

Это стартовая версия нейромагазина на базе ChatGPT API.

## Как использовать

1. Вставьте ваш OpenAI API ключ в файл `ai-assistant.js`, заменив `YOUR_OPENAI_API_KEY`.
2. Загрузите проект на GitHub Pages или локальный сервер.
3. Вводите запросы в интерфейсе — ассистент подберёт лучшие предложения.

## Примеры запросов

- "беспроводные наушники до 50 евро"
- "монитор 24 дюйма до 3000 Kč"
- "подарок девушке до 30 евро"

Проект использует HTML + JS и подходит для запуска минимального MVP.
